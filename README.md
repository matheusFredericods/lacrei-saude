
# 💚 Lacrei Saúde — Desafio Front-End

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da **Lacrei Saúde**, com foco em criar uma aplicação web moderna, acessível e responsiva, seguindo o **Marsha Design System**.

A aplicação simula um ambiente real de produção, priorizando boas práticas de desenvolvimento Front-End, acessibilidade, performance e organização de código.

---

## 🎯 Objetivo

Desenvolver uma interface com:

* Fidelidade ao Design System da Lacrei Saúde
* Acessibilidade como prioridade
* Responsividade (mobile-first)
* Performance otimizada
* Estrutura escalável e organizada

---

## 🚀 Tecnologias Utilizadas

* Next.js (App Router)
* TypeScript
* Styled-Components
* React
* Next Image (otimização de imagens)

---

## 📄 Páginas do Projeto

A aplicação possui 3 páginas principais:

* 🏠 **Home** — Página inicial com apresentação do serviço
* 👥 **Quem Somos** — Informações sobre a missão e propósito
* 📅 **Agendamento** — Página para simulação de agendamento

---

## 🧭 Componentes Principais

* Header personalizado com navegação entre páginas
* Footer personalizado com informações institucionais
* Botões interativos com ações reais
* Componentes reutilizáveis para melhor escalabilidade

---

## 🔘 Interatividade

O projeto conta com **botões funcionais**, incluindo:

* Navegação entre páginas
* Ações interativas na interface
* Elementos com foco em acessibilidade

---

## 📱 Responsividade

O layout foi desenvolvido com abordagem **mobile-first**, garantindo adaptação para:

* 📱 Mobile
* 💻 Tablet
* 🖥️ Desktop

---

## ♿ Acessibilidade

A acessibilidade foi tratada como pilar principal:

* HTML semântico (`header`, `main`, `footer`)
* Uso de `aria-label` quando necessário
* Estrutura compatível com leitores de tela
* Contraste adequado seguindo boas práticas
---

## ⚡ Performance

O projeto foi otimizado para performance:

* Uso de `next/image` para otimização de imagens
* Componentização para melhor renderização
* Estrutura leve e eficiente
* Objetivo de atingir boa nota no Lighthouse

---

## 🧪 Estrutura de Testes

Testes unitários foram aplicados em componentes essenciais:

* Header
* Footer
* Botões interativos

Ferramentas utilizadas:

* Jest
* React Testing Library

## 🚀 Como rodar o projeto localmente

```bash
# Clonar o repositório
git clone https://github.com/matheusFredericods/lacrei-saude.git

# Acessar a pasta
cd lacrei-saude

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```

---

## 🏗️ Build de Produção

```bash
npm run build
npm start
```
